# GitHub Upload Guide

```bash
git init
git add .
git commit -m "Initial commit: AI enterprise audit invoice compliance platform"
git branch -M main
git remote add origin https://github.com/<your-username>/ey-ai-enterprise-audit-invoice-compliance.git
git push -u origin main
```

Recommended repository description:

> Clean-room AI enterprise invoice validation, audit compliance, fraud detection, RAG policy decisioning, and multi-agent workflow automation platform.

Suggested topics:

```text
fastapi langgraph rag invoice-processing fraud-detection audit-compliance prometheus grafana fintech ai-agents document-intelligence
```

Before publishing, replace any resume-specific company wording with "enterprise audit demo" unless you are explicitly presenting it as a portfolio simulation.
